# V1.2 CompositionBalance Tuning and Marketplace-card Visual Reranking

## What changed
- V1.2 fixes a real visual-geometry bug in the visual model: text blocks with `h = 0` were effectively invisible to `compositionBalance`, which made compact marketplace-card stacks look like accidental corner-collapse.
- The visual layer now estimates text box height from font size, line count, and format height before scoring visual balance and harmony.
- `marketplace-card` visual reranking is now slightly stronger in production, but still only inside safe structural windows.

## CompositionBalance tuning
- `compositionBalance` now uses estimated text bounds instead of trusting zero-height title/subtitle boxes.
- Marketplace compact-stack moderation now triggers from headline/text placement rather than the full CTA-skewed text cluster.
- This reduces false corner-collapse penalties for valid compact marketplace layouts without inflating invalid layouts globally.

## Calibration result
- The V1.1 noisy calibration cases for valid marketplace outputs are no longer flagged as noisy.
- Representative shift:
  - `marketplace-card` fresh no-image: visual `44 / poor` -> `63 / weak`
  - `marketplace-highlight` fresh no-image: visual `45 / poor` -> `61 / weak`
- Invalid controls remain suppressed:
  - `social-square`: still `53 / weak`

## Stronger marketplace-card visual reranking
- Preview-path visual reranking remains enabled only for `marketplace-card` and only when structural tier is equal, candidates are not invalid, and effective scores are close.
- In the 300-context preview sample this still changed `0` final winners, which means preview selection was already structurally dominant.
- The meaningful user-visible effect now appears in `marketplace-card` repair winner selection:
  - no-image accepted repair attempts now apply visual reranking before lower-level repair tie-breaks
  - the first accepted repair winner for the fresh default no-image case moves from bland `text-stack` to stronger `split-vertical`

## Why the scope stays narrow
- Structural tier remains first.
- Invalid candidates still cannot win on visual score alone.
- Global ranking, validation, and non-marketplace-card production behavior stay unchanged.
